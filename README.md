# Hide X Sidebars Toggle

![Tree icon](./assets/tree-icon.svg)

A simple Chrome extension for `x.com` and `twitter.com` that fades the left and right sidebars in and out with a single toggle button.

The toggle is a tree icon fixed to the bottom-left corner of the page:

- When enabled, both sidebars fade out immediately.
- When disabled, both sidebars fade back in.
- The tree icon matches X's theme by appearing black in light mode and white in dark mode.
- The icon animates by growing leaves when enabled and reversing that motion when disabled.

## Load locally

1. Open `chrome://extensions`
2. Turn on `Developer mode`
3. Click `Load unpacked`
4. Select this folder
